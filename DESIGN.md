# Kindred Design System (Mobile Edition)

This document outlines the final visual identity and design tokens for the **Kindred** mobile application.

## Creative North Star: "The Neural Nexus"
Kindred is designed as a living intelligence layer—a "Neural Nexus" that feels both mathematically precise and humanly intuitive. It moves away from rigid SaaS layouts toward a fluid, editorial experience optimized for networking events.

## Color Palette

| Category | Token | Value | Description |
| :--- | :--- | :--- | :--- |
| **Primary** | `primary` | `#cc97ff` | Represents "Intelligence". Used for core actions and paths. |
| **Secondary** | `secondary` | `#3adffa` | Represents "Serendipity". Used for high-value matches and nodes. |

### Atmospheric Strategy
- **No-Line Rule**: Boundaries are defined by background color shifts (tonal hierarchy) rather than 1px borders.
- **Tonal Depth**: Treat the UI as stacked, semi-translucent layers using `surface-container-low` to `surface-container-highest`.
- **Signature Gradient**: Linear 135° from `primary` (#7846a8) to `primary_container` (#cc97ff) for hero CTAs.

## Typography

| **Body & Labels** | **Inter / Manrope** | High legibility for data-heavy networking stats. |

### Scale Hierarchy
- Use extreme scale differences (e.g., `display-lg` vs `label-sm`) to create an "Infographic" narrative within the individual profile views.

## Component Specifications

### 1. The Network Node
A circular avatar with a **Cyan (#3adffa)** "connectivity ring." The ring’s thickness or pulse animation indicates the strength of the networking "intelligence" match.

### 2. Serendipity Score Badge
A large, glowing cyan circle containing a numerical score (e.g., "88"). This is the primary indicator of connection quality.

### 3. Micro-Circle Triangle
A specialized component for group matches. Three avatars arranged in a triangle formation with directional "Ask/Offer" arrows connecting them.

## Elevation & Depth
- **Corner Roundness**: `ROUND_EIGHT` (12px / 1rem approx) for all cards and primary buttons.
- **Ambient Shadows**: Extra-diffused, tinted shadows (Blur: 32px+) with low opacity (4-6%) using a tinted version of the primary color.
